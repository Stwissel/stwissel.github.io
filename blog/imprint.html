Tools I use

  • This blog serves static pages from an nginx HTTP server running on Ubuntu
  • Comments are captured in a CouchDB database and periodically added to the static pages
    • Comment entry is based on Markdown by John Gruber
    • JavaScript uses the AngularJs framework
  • Graphics are created with SmartDraw, the only other software besides Domino Designer I run on Windows
  • Image manipulation is done by The Gimp
  • The mindmaps are either iMindMap by Tony Buzan or XMind, the living prove, that Eclipse RCP applications can perform well
  • Layout is based on Twitter Bootstrap
