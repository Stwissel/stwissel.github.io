Tools I use

  • This blog serves static pages from an nginx HTTP server running on Ubuntu
  • Blog articles are created using Markdown and stored in a Bitbucket repository. On new content creation a Bitbucket pipeline renders the blog entriy and index pages using Java and Mustache
  • Comments are captured using a Heroku application that submits a file to Bitbucket and creates a Pull Request that I need to approve (contemplating to automate that) to be added to the static pages
    • Comment entry is based on Markdown by John Gruber
    • JavaScript uses the AngularJs framework
  • Graphics are created with SmartDraw
  • Wireframes are created with Balsamiq Mockups
  • Sequence diagrams use WebSequenceDiagrams
  • Image manipulation is done by The Gimp
  • The mindmaps are either iMindMap by Tony Buzan or XMind, the living prove, that Eclipse RCP applications can perform well
  • Layout is based on Twitter Bootstrap
